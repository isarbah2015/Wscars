Add a new "Car Genomics" infographic feature to the Westcars app. This feature shows aggregated review data for each car model with beautiful visualizations.

## What to create:

### 1. Update Firestore Structure
- Add a `reviews` collection (if not exists) with fields: `carId`, `userId`, `rating` (1-5), `comment`, `aspectRatings` (object with comfort, performance, safety, interiorQuality, value)
- Add a `carGenomics` collection to store pre-aggregated stats per car model

### 2. New Component: `src/components/CarGenomics.js`
Create a component that displays:

- Overall average rating (big number with stars)
- Horizontal bar charts for each aspect (Comfort, Performance, Safety, Interior Quality, Value)
- Top 5 positive keywords from reviews (e.g., "spacious", "smooth")
- Top 5 negative keywords (e.g., "noisy", "small trunk")
- Total review count badge

### 3. Visual Style:
- Use orange (#FF6B00) for filled bars and accents
- Glassmorphic card design matching the app
- Bars should be animated when loading
- Support dark/light mode
- Use any chart library (Victory Native, react-native-svg-charts, or react-native-gifted-charts)

### 4. Integration
- Add a new "Genomics" tab on the Car Detail Screen (next to Specifications)
- When tapped, fetch data from Firestore `carGenomics` collection using the car model as key
- Show loading state, then render the infographic
- If no reviews exist, show "Be the first to review this car"

### 5. Cloud Function (optional but recommended)
Create a Firebase Cloud Function `calculateCarGenomics` that:
- Runs whenever a new review is added
- Recalculates averages and keyword frequencies for that car model
- Updates the `carGenomics` document

### 6. Mock Data for Testing
Include hardcoded example data for a few car models so the feature works immediately without waiting for real reviews.

## Files to create/update:
- `src/components/CarGenomics.js`
- `src/screens/CarDetailScreen.js` (add Genomics tab)
- `src/utils/genomicsData.js` (mock data helper)
- `firebase/functions/index.js` (cloud function, if using)

Generate all code with comments. Make the infographic look modern and professional.
